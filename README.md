# ScamShield AI

##  Overview
ScamShield AI is a smart web-based tool that detects scam and phishing messages using rule-based AI logic and keyword analysis.

## 🧠 How it Works
1. User pastes an SMS message
2. System analyzes keywords like:
   - bank, account, card
   - urgent, locked, verify
   - win, prize, congratulations
3. It assigns a risk score (0–100)
4. Classifies message type:
   - Bank Phishing Scam
   - Lottery Scam
   - Phishing Scam
   - Safe Message
5. Displays result with detailed breakdown

## 🤖 AI Concept (Google Antigravity)
The system simulates an AI agent workflow:
- Insight Extraction Agent
- Risk Scoring Engine

## 🎯 Features
- Instant SMS scam detection
- Risk scoring system
- Scam type classification
- Agent-style explanation output

## 📌 Purpose
To help users identify scam messages and improve digital safety awareness.
